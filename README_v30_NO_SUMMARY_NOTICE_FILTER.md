# v30 변경사항

## 반영 내용

1. 기사 요약보기 기능 폐기
   - 원문 열기 옆 `기사 요약 보기` 버튼 제거
   - 요약 팝업 및 보드판 이미지 리소스 제거
   - 기사요약 컬럼은 과거 Supabase/CSV 호환을 위해 내부 컬럼으로만 유지하며 화면에는 표시하지 않음

2. 경조사/인사/동정성 기사 제외
   - 화촉, 부음, 부고, 인사, 동정, 사령, 결혼, 별세, 부친상, 모친상, 빙부상, 빙모상 등 포함 기사 제외
   - 언론사명만 반복되는 목록형 기사 예: `데일리팜 - 데일리팜` 제외
   - 기존 캐시에 남아 있어도 앱 로드/재분류 시 화면에서 제외

3. 분류 기준 정리
   - 분류 판단은 제목 + RSS 요약 + 원문 본문 일부를 사용
   - v29의 요약문은 분류 근거에서 제외하여 요약 실패 문구가 분류에 섞이지 않도록 수정

git commit -m "Remove article summary popup and filter notice-type news"
