제약뉴스 RSS 대시보드 v1.11

실행방법
1. 압축을 새 폴더에 해제합니다.
2. run_dashboard.bat 을 더블클릭합니다.
3. 브라우저에서 대시보드가 열립니다.

이번 버전 목적
- 원문 링크 버튼이 안 보이는 원인을 단계적으로 확인하기 위한 진단 버전입니다.
- 화면 상단에 Version: v1.11 문구가 표시되어야 이번 버전이 정상 실행 중입니다.
- 조회조건 아래의 '링크 진단 / v1.11' 영역을 열면 link 컬럼 보유 현황을 확인할 수 있습니다.

중요 수정
- v1.9까지 link 컬럼에 일반 텍스트 정리 로직이 적용되어 URL 문자열이 제거되는 문제가 있었습니다.
- v1.11부터 link는 URL 보존 전용 clean_link() 로직으로 처리합니다.
- 기존 news_clean.csv에 이미 link가 삭제된 기사는 자동 복구가 어렵습니다.
- 이 경우 RSS 수집 버튼을 다시 눌러 신규 또는 동일 기사를 재수집하면 link가 있는 행을 우선 보존하도록 수정했습니다.

확인 순서
1. 화면 상단 Version 문구 확인
2. 링크 진단 패널에서 link 보유 건수 확인
3. 카드에서 '원문 열기 ↗' 또는 '링크 없음' 배지가 보이는지 확인
4. link 보유 건수가 0이면 RSS 수집을 다시 실행
